## Prinsip Dasar
Project membedakan penjualan online dan offline sehingga pengurangan stok dan pengakuan transaksi tidak boleh selalu dianggap terjadi di titik yang sama.

## Rule Utama
- status final / completed harus mengurangi stok,
- status yang belum final tidak boleh mengurangi stok permanen,
- cancel / delete harus mengembalikan stok bila sudah pernah dipotong.
