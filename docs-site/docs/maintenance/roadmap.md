# Roadmap

Berikut arah pengembangan yang paling masuk akal setelah kondisi file terbaru.

## Prioritas dekat

### 1. Integrasi payroll yang lebih dalam ke costing produksi

Payroll produksi sudah ada, tetapi berikutnya bisa dibuat lebih menyatu dengan laporan biaya dan analisis lanjutan.

### 2. Pendalaman HPP produk jadi

Analisis HPP produksi sudah berjalan untuk work log completed. Tahap berikutnya adalah memastikan hasil HPP lebih mudah diturunkan ke produk jadi dan pricing.

### 3. Integrasi fee marketplace yang lebih menyeluruh

Pricing rules sudah mendukung marketplace buffer. Tahap berikutnya adalah menyambungkan fee aktual marketplace ke analisis yang lebih lengkap.

## Prioritas menengah

### 4. Penyempurnaan penyesuaian stok

Menu penyesuaian stok masih cocok untuk koreksi manual. Ke depan, flow ini bisa makin diselaraskan dengan arsitektur stok terbaru agar lebih aman untuk semua tipe item.

### 5. Dokumentasi menu yang makin detail

Dokumentasi dapat diperluas lagi sampai level:

- field per form,
- status per transaksi,
- contoh skenario user,
- dan checklist testing per menu.

### 6. Penyatuan style seluruh sistem

Bahasa desain antar halaman sudah jauh lebih rapi, tetapi masih bisa diteruskan ke:

- detail drawer,
- form modal,
- status tags,
- card statistik,
- dan spacing layout lintas menu.
