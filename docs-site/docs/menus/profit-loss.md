# Laba Rugi

## Tujuan
Dipakai untuk membaca performa usaha secara ringkas.

## Komponen Bacaan
- pendapatan,
- biaya pembelian atau produksi,
- biaya operasional,
- pemasukan lain,
- pengeluaran lain,
- laba atau rugi periode.

## Contoh Skenario
Cek laba rugi bulanan untuk melihat apakah margin masih sehat setelah biaya produksi dan operasional dihitung.
