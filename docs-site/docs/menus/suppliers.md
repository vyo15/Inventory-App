# Supplier

## Tujuan
Menu ini dipakai untuk mencatat supplier beserta bahan yang tersedia dan link produk belanja.

## Informasi di Tabel
- nama supplier,
- kategori atau keterangan supplier,
- jumlah material yang tersedia,
- aksi edit dan hapus.

## Field Form Utama
- **Kategori / Keterangan Supplier**
- **Nama Supplier / Toko**
- **Link Toko**
- **Material Detail** per baris:
  - Bahan,
  - Link Produk,
  - Harga Catatan,
  - Catatan.

## Tombol Aksi
- **Tambah Supplier**
- **Edit**
- **Hapus**
- **Tambah Material**
- **Hapus Baris**

## Contoh Skenario
Simpan 1 supplier kain flanel, lalu tambahkan beberapa link produk warna berbeda agar tim belanja lebih cepat saat restock.
