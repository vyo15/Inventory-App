# Laporan Penjualan

## Tujuan
Dipakai untuk merangkum transaksi penjualan per periode.

## Yang Umum Dicek
- total penjualan,
- customer atau channel paling aktif,
- item yang paling laku,
- status penjualan.

## Contoh Skenario
Gunakan untuk melihat produk yang paling sering terjual sebelum menentukan prioritas produksi.
