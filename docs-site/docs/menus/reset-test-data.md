# Reset Data Uji

## Tujuan
Dipakai untuk membersihkan data simulasi, menyimpan baseline, dan sinkronisasi stok.

## Tombol Utama
- **Refresh Preview**
- **Simpan Baseline**
- **Sinkronkan Stok**
- **Reset Sekarang**

## Konfirmasi
Sebelum reset terakhir, user harus mengetik **RESET**.

## Yang Perlu Dipahami
- reset dipakai untuk testing, bukan operasi harian,
- **Sinkronkan Stok** merapikan field stok seperti current, reserved, available, dan total,
- reset tidak otomatis membetulkan mapping logic PO atau BOM yang salah; itu harus dibetulkan dari data atau logic-nya.

## Contoh Skenario
Setelah banyak simulasi produksi, jalankan preview, pilih modul yang ingin dibersihkan, lalu reset. Setelah itu klik **Sinkronkan Stok** untuk merapikan pembacaan stok.
