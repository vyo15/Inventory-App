# Kategori

## Tujuan
Menu ini dipakai untuk menyimpan daftar kategori yang dipakai produk atau bahan baku.

## Field Form
- **Nama Kategori**
- **Deskripsi**

## Tombol Aksi
- **Tambah Kategori**
- **Edit**
- **Hapus**

## Catatan
Kategori masih aman dihapus jika belum dipakai data lain. Jika sudah banyak dipakai, lebih aman rapikan nama kategori daripada sering hapus dan buat ulang.

## Contoh Skenario
Buat kategori seperti **Bunga**, **Bahan Pelengkap**, atau **Kemasan** agar filter menu master data lebih rapi.
