# Transaksi

## Tujuan
Kelompok menu ini dipakai untuk transaksi harian yang mengubah stok atau nilai bisnis.

## Menu di Dalamnya
- **Pembelian**: stok masuk dari supplier.
- **Penjualan**: stok keluar ke customer.
- **Retur**: koreksi balik stok dari barang yang diretur.

## Rule Penting
- pembelian menambah stok,
- penjualan mengurangi stok,
- retur dipakai untuk mengembalikan stok sesuai kasus yang berlaku.
