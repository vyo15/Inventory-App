# Laporan Stok

## Tujuan
Dipakai untuk melihat posisi stok seluruh item dalam format laporan.

## Informasi di Tabel
- nama item,
- kategori,
- jenis item,
- stok,
- satuan,
- status.

## Filter
- pencarian item,
- filter jenis dan status,
- tombol export bila tersedia.

## Contoh Skenario
Cek item mana yang kosong atau menipis sebelum menyusun restock atau produksi mingguan.
