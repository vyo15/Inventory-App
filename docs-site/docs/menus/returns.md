# Retur

## Tujuan
Dipakai untuk mencatat pengembalian barang dan menyesuaikan stok sesuai kebijakan retur yang berlaku.

## Field Form
- **Tanggal**
- **Jenis Item**
- **Nama Item**
- **Jumlah**
- **Catatan**

## Tabel Utama
Menampilkan tanggal, jenis, nama item, jumlah, dan catatan.

## Contoh Skenario
Jika ada barang yang kembali dari transaksi tertentu, input retur agar stok dan histori transaksi tetap jelas.
