# Pengeluaran

## Tujuan
Dipakai untuk mencatat pengeluaran operasional non-pembelian utama.

## Field Form
- **Tipe Pengeluaran**
- **Jumlah**
- **Deskripsi**
- **Tanggal**

## Tabel Utama
Menampilkan tanggal, tipe, deskripsi, aktual keluar, total referensi, saving / selisih, supplier, dan aksi.

## Tombol Aksi
- **Tambah Pengeluaran**
- **Hapus**

## Contoh Skenario
Catat biaya listrik, transport, atau biaya admin yang tidak masuk menu Pembelian.
