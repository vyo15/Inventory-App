# Pelanggan

## Tujuan
Menu ini dipakai untuk menyimpan daftar customer agar input penjualan lebih cepat dan rapi.

## Field Form
- **Nama Customer**
- **Kontak**
- **Alamat**
- **Catatan**

## Tombol Aksi
- **Tambah Customer**
- **Edit**
- **Hapus**

## Contoh Skenario
Simpan customer reseller atau pelanggan tetap yang sering order agar transaksi penjualan tinggal pilih nama customer.
