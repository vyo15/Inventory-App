# Inventory

## Tujuan
Kelompok menu ini dipakai untuk melihat log stok dan melakukan penyesuaian stok.

## Menu di Dalamnya
- **Manajemen Stok**: histori mutasi stok dari pembelian, penjualan, produksi, dan penyesuaian.
- **Penyesuaian Stok**: koreksi stok jika ada selisih fisik.

## Rule Penting
- log stok adalah jejak audit, jadi sebaiknya tidak dihapus,
- penyesuaian stok dipakai untuk koreksi, bukan untuk transaksi rutin.
