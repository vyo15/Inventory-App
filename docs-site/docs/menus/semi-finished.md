# Bahan Setengah Jadi

Menu ini dipakai untuk menyimpan hasil antara dari proses produksi, misalnya komponen atau potongan yang belum menjadi produk jadi.

## Tujuan

- memisahkan bahan mentah dan hasil antara
- memudahkan BOM bertingkat
- memantau output antar tahap produksi

## Yang ditampilkan

- nama item
- kategori
- total stok
- rincian varian
- status stok
- aksi detail dan edit

## Rule penting

- output work log akan masuk ke item ini jika target type adalah semi finished
- jika item punya varian, output masuk ke varian target
