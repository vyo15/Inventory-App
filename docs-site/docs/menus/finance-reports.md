# Kas dan Biaya

## Tujuan
Kelompok menu ini dipakai untuk mencatat pemasukan tambahan dan pengeluaran di luar transaksi utama.

## Menu di Dalamnya
- **Pemasukan**: pendapatan lain-lain atau pemasukan non-penjualan.
- **Pengeluaran**: biaya operasional atau pengeluaran lain di luar pembelian utama.

## Rule Penting
- jangan campurkan pemasukan atau pengeluaran operasional dengan transaksi penjualan atau pembelian utama bila datanya sudah punya menu sendiri.
