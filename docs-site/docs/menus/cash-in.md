# Pemasukan

## Tujuan
Dipakai untuk mencatat pemasukan non-penjualan.

## Field Form
- **Tipe Pemasukan**
- **Jumlah**
- **Deskripsi**
- **Tanggal**

## Tabel Utama
Menampilkan tanggal, jumlah, tipe, sumber data, deskripsi, dan aksi.

## Tombol Aksi
- **Tambah Pemasukan**
- **Hapus**

## Contoh Skenario
Catat pemasukan lain-lain seperti penjualan aset kecil atau pendapatan tambahan yang tidak dicatat lewat menu Penjualan.
