# Dokumentasi IMS Bunga Flanel

Dokumentasi ini dibuat untuk membantu proses pengembangan, maintenance, audit alur bisnis, dan sinkronisasi logic sistem pada project IMS Bunga Flanel.

## Tujuan Dokumentasi
- Menjelaskan alur bisnis dan alur sistem
- Menjelaskan fungsi tiap menu
- Menyimpan aturan logic penting
- Menandai fitur yang sudah diperbaiki
- Menjadi panduan maintain project jangka panjang

## Cakupan
Dokumentasi ini mencakup:
- Overview project
- Business flow
- System logic
- Dokumentasi per menu
- Improvement log
- Known issues
- Roadmap
