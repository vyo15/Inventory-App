# Project Overview

IMS Bunga Flanel adalah sistem inventory dan operasional untuk mengelola bahan baku, bahan setengah jadi, produk jadi, pembelian, produksi, penjualan, laporan, dan pengembangan pricing rules.

## Tujuan Sistem
Sistem ini dibuat agar:
- stok bahan baku tercatat rapi
- proses produksi dapat dilacak
- stok semi finished dan finished goods terhubung
- penjualan online dan offline dapat dibedakan
- laporan keuangan dan operasional lebih mudah dibaca

## Karakter Sistem
Sistem ini berfokus pada:
- pencatatan stok
- alur produksi bertahap
- pengendalian pembelian
- keterhubungan data antar menu
- maintainability jangka panjang
