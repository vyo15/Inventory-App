## Frontend
Project utama dibangun dengan:
- **React**
- **Vite**
- **React Router**
- **Ant Design**

## Backend / Database
Penyimpanan data dan sinkronisasi menggunakan:
- **Firebase**
- **Cloud Firestore**

## Dokumentasi
Dokumentasi internal project dibuat dengan:
- **Docusaurus**

## Karakter Stack
Stack ini dipilih karena:
- cepat untuk pengembangan UI dan admin panel,
- cocok untuk CRUD dan realtime-ish workflow,
- mudah dipecah per menu dan service,
- cukup fleksibel untuk project yang masih berkembang.
