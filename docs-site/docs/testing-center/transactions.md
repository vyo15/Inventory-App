# Checklist Transaksi

Checklist ini dibuat dari modul transaksi aktif:
- Purchases
- Sales
- Returns
- Cash In
- Cash Out

## Purchases
- [ ] Purchase baru berhasil dibuat
- [ ] Supplier tampil benar
- [ ] Raw material tampil benar
- [ ] Qty dan subtotal tampil benar
- [ ] Actual price tersimpan benar
- [ ] Saving tampil sebagai efisiensi, bukan pengurang kas
- [ ] Stock raw material bertambah sesuai qty

## Sales
- [ ] Sales baru berhasil dibuat
- [ ] Customer tampil benar
- [ ] Product / variant tampil benar
- [ ] Status transaksi terbaca benar
- [ ] Completed mengurangi stok
- [ ] Cancel / delete mengembalikan stok sesuai rule
- [ ] Nilai transaksi sinkron ke report sales

## Returns
- [ ] Return bisa dibuat
- [ ] Dampak stok return sesuai rule
- [ ] Dampak laporan return sesuai rule

## Cash In
- [ ] Data pemasukan tampil
- [ ] Tambah pemasukan normal
- [ ] Data terbaca di laba rugi bila memang masuk alur

## Cash Out
- [ ] Data pengeluaran tampil
- [ ] Tambah pengeluaran normal
- [ ] Data terbaca di laba rugi
