# UAT Final

Halaman ini dipakai untuk validasi akhir sebelum milestone besar, push penting, atau deploy.

## Validasi Umum
- [ ] Semua menu utama bisa dibuka
- [ ] Tidak ada error besar di console
- [ ] Form utama bisa simpan
- [ ] Data bisa reload setelah simpan

## Validasi Flow Inti
- [ ] Purchase → stock raw material
- [ ] Produksi → input berkurang, output bertambah
- [ ] Sales → stock product berkurang
- [ ] Reports → membaca data transaksi terbaru

## Validasi Data
- [ ] Variant stock sinkron
- [ ] currentStock sinkron
- [ ] Harga / cost penting tidak kosong

## Validasi Dokumentasi
- [ ] Intro docs tidak bentrok lagi
- [ ] Bug fix penting sudah dicatat
- [ ] Known issues sudah diperbarui
- [ ] Testing Center diperbarui
