# Known Issues Produksi

## Hal yang Perlu Diwaspadai
- PO lama bisa menyimpan requirement snapshot lama,
- perubahan BOM setelah PO dibuat tidak otomatis mengubah PO lama,
- item bervarian harus dicek apakah source bahan sudah membaca varian yang benar, bukan master.

## Saran Operasional
- setelah perubahan besar BOM atau varian, buat PO baru,
- cek detail PO sebelum mulai produksi,
- gunakan menu log stok untuk verifikasi potongan bahan.
