# Analisis HPP

Menu ini dipakai untuk membaca biaya produksi yang sudah terjadi.

## Komponen yang dianalisis

- biaya material
- biaya tenaga kerja
- output bagus
- biaya per unit
- total biaya per work log

## Tujuan

- audit biaya realisasi
- lihat biaya per hasil produksi
- bantu evaluasi harga dan efisiensi

## Catatan

Analisis HPP adalah hasil baca data, bukan tempat mengubah histori stok atau payroll.
