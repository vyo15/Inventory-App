# Analisis HPP

## Tujuan
Dipakai untuk membaca biaya aktual per work log dan menghitung HPP per unit.

## Informasi di Tabel
- no. work log,
- target / step,
- good qty,
- material cost,
- direct labor,
- overhead,
- total cost,
- HPP per unit.

## Filter
- pencarian work log, target, atau step,
- tombol **Refresh**.

## Cara Membaca
- fokus ke **Good Qty** karena HPP per unit dibagi ke output bagus,
- jika good qty kecil, HPP per unit akan naik,
- data payroll dan biaya lain akan mempengaruhi hasil HPP.

## Contoh Skenario
Setelah beberapa work log selesai, buka menu ini untuk melihat apakah HPP bunga mawar masih sesuai target harga jual.
