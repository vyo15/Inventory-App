# Alur Aksi Produksi

## Aksi 1: Buat PO
- pilih BOM,
- pilih varian target jika perlu,
- isi qty batch,
- simpan.

## Aksi 2: Mulai Produksi
- cek requirement material,
- klik **Mulai Produksi**,
- sistem membuat work log dan memotong bahan.

## Aksi 3: Selesaikan Work Log
- isi qty bagus, reject, rework,
- pilih operator,
- isi catatan bila perlu,
- klik **Selesaikan**.

## Aksi 4: Buat Payroll
- pilih work log completed,
- pilih karyawan,
- cek hasil hitung,
- simpan atau tandai paid.
