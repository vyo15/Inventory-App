# Status dan Transisi Produksi

## Production Order
- **draft / ready / shortage**
- **in_production**
- **completed**
- **cancelled**

## Work Log
- **draft**
- **in_progress**
- **completed**

## Payroll
- **draft**
- **paid**

## Catatan
Transisi utama yang harus user pahami:
- PO dibuat -> stok belum berubah,
- PO mulai produksi -> work log dibuat dan bahan dipotong,
- work log selesai -> output bertambah dan PO ditutup.
