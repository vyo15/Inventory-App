# Overview Produksi

## Tujuan
Bagian produksi dipakai untuk mengubah bahan baku menjadi semi finished atau produk jadi secara terstruktur.

## Flow Singkat
1. siapkan master produksi,
2. buat BOM,
3. buat PO,
4. mulai produksi,
5. selesaikan work log,
6. hitung payroll dan baca HPP.

## Prinsip Penting
- input dipotong saat PO mulai dikerjakan,
- output ditambah saat work log selesai,
- varian target harus konsisten agar source bahan terbaca benar.
