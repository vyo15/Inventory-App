# Referensi Field Produksi

## Istilah Kunci
- **Qty Batch**: jumlah batch yang dikerjakan.
- **Hasil per Produksi / Batch Output Qty**: output standar dari 1 batch pada BOM.
- **Estimasi Output**: hasil per produksi x qty batch.
- **Good Qty**: output bagus yang jadi stok.
- **Reject Qty**: output gagal.
- **Rework Qty**: output yang perlu dikerjakan ulang.
- **Requirement Material**: daftar bahan yang dibutuhkan PO.
- **Variant / Sumber**: varian stok yang dipakai bahan.
- **Average Cost**: rata-rata biaya per unit pada item.

## Tips Baca
Jika angka **Estimasi Output** terasa salah, cek dulu nilai **Hasil per Produksi** di BOM dan **Qty Batch** di PO.
