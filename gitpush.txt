IMS SOURCE ZIP — COMPATIBILITY NOTE
===================================

File ini dipertahankan agar shortcut/catatan lama tidak langsung hilang saat project dipakai
bergantian di laptop dan PC kantor. Script PowerShell hardcoded lama sudah tidak digunakan.

Buat ZIP source resmi dari ROOT project dengan salah satu command berikut.

Git Bash:
  npm run clean:zip

PowerShell:
  npm run clean:zip:ps

Verifikasi ZIP yang dihasilkan:
  npm run verify:zip -- ../NAMA-FILE-CLEAN.zip

Jangan membuat source ZIP dengan Compress/Send to ZIP dari File Explorer karena folder runtime
`data/`, `backups/`, `logs/`, node_modules, build output, dan file lokal lain dapat ikut terbawa.

Workflow Git tetap manual agar aman saat bekerja dari dua perangkat:
  git status --short
  git pull --rebase origin main
  git add -A
  git status
  git commit -m "pesan perubahan"
  git push origin main
