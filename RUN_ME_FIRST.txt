IMS Bunga Flanel - paket project siap jalan

Langkah:
1. Extract zip
2. Buka folder project
3. Jalankan: npm install
4. Jalankan: npm run dev
5. Jika browser masih baca cache lama, tekan Ctrl+Shift+R

Catatan:
- package.json sudah disesuaikan agar perintah Vite lebih aman dijalankan lintas environment
- build sudah diverifikasi berhasil di container sebelum zip ini dibuat
